## 프로젝트 주제 및 목적
* 칸반 보드 기반의 협업 툴 구현
* JavaMailSender를 활용한 이메일 전송 구현
* Interceptor를 활용한 로그인 처리
* Jasypt를 활용한 암, 복호화 구현
* Chart.js를 활용한 데이터 시각화
* SortableJS를 활용한 Drag&Drop 구현
* LocalStorage를 활용한 TodoList 구현
* Oracle Cloud를 활용한 DB 배포
